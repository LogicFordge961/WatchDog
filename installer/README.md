# WatchDog - System Intelligence Framework

## Features
- 🤖 AI Assistant for system diagnostics
- 📊 Process monitoring and management
- 🔧 Application launcher and scanner
- 🔒 Security monitoring
- 🔄 Auto-update system

## Installation
1. Run the installer
2. Launch WatchDog from Start Menu
3. Type 'help' for commands

## Commands
- ai <question> - Ask AI about system performance
- kill <process> - Terminate processes
- scan - Scan for installed applications
- monitor - Start system monitoring
- help - Show help

## Requirements
- Windows 10/11
- Python 3.7+
- Internet connection for AI features
